# OMR Grader — Release Channel

ช่องแจกอัปเดตสาธารณะของโปรแกรมตรวจกระดาษคำตอบ OMR
ไฟล์ติดตั้งและอัปเดตทั้งหมดอยู่ที่แท็บ **Releases**

- `OMR_Grader_Setup.exe` — ตัวติดตั้งสำหรับครู (ติดตั้งได้โดยไม่ต้องใช้สิทธิ์ admin)
- `version.json` — manifest สำหรับระบบอัปเดตอัตโนมัติ (เซ็นลายเซ็น)
- `build-manifest.txt` — บันทึกว่าเวอร์ชันนั้น build ด้วยไลบรารีชุดไหน

> โค้ดต้นฉบับอยู่ในที่เก็บส่วนตัวแยกต่างหาก
