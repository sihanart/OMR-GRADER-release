OMR Grader

โปรแกรมตรวจกระดาษคำตอบอัตโนมัติ · แม่นยำ · ใช้ง่าย · ฟรี

⬇ ดาวน์โหลดโปรแกรม (Windows) 📄 โหลดคู่มือ (PDF)

ไฟล์เดียว ดับเบิลคลิกใช้ได้เลย — ไม่ต้องติดตั้ง Python · รองรับฟอร์ม R2202A (60 ข้อ) และ R1102 (120 ข้อ)

เริ่มใช้ใน 5 ขั้นตอน

เตรียมกระดาษ: ให้นักเรียนฝนด้วยดินสอ 2B เข้มๆ แล้วสแกนเป็นรูป (200–300 DPI) ให้ตรง ไม่เอียง

1

เลือกฟอร์ม

เลือก R2202A หรือ R1102 ให้ตรงกับกระดาษ

2

ตั้งเฉลย

เร็วสุด: ฝนเฉลยลงกระดาษเปล่า 1 ใบ แล้วสแกน — โปรแกรมอ่านเป็นเฉลยให้เอง (หรือกรอกทีละข้อก็ได้)

3

เพิ่มรูป

กด “เพิ่มรูป” เลือกไฟล์สแกน ทีละหลายใบได้

4

ตรวจ

กด “ตรวจทั้งหมด” — เขียว = ถูก, แดง = ผิด

5

บันทึกผล

ส่งออก Excel 3 ชีต (คะแนน · วิเคราะห์รายข้อ · สรุป+กราฟ) หรือ CSV

⚠ ครั้งแรกที่เปิด Windows อาจเตือน “Windows protected your PC” เพราะไฟล์ยังไม่ได้เซ็น certificate → กด More info แล้วกด Run anyway ครั้งเดียว ครั้งต่อไปจะไม่ถามอีก (โปรแกรมปลอดภัย)

ความสามารถเด่น

ครบเครื่องสำหรับงานตรวจข้อสอบจริง

🎯

แม่นระดับพิกเซล

ปรับมุมภาพอัตโนมัติ อ่านตำแหน่งวงจาก template จริง

🔢

อ่านเลขที่นั่งสอบ

ดึงเลขที่นั่งสอบ/รหัสวิชาจากที่ฝนให้อัตโนมัติ ลง CSV

📊

รายงาน Excel 3 ชีต

คะแนนรายคน · วิเคราะห์รายข้อ (ข้อไหนเด็กพลาดเยอะ) · สรุป+กราฟเกรด

📄

เฉลยจากการสแกน

ฝนเฉลยลงกระดาษ 1 ใบแล้วสแกน ไม่ต้องกรอกทีละข้อ

🛡️

กันแก้คะแนน

ไฟล์ผลมีลายเซ็นดิจิทัล ตรวจจับการแก้ไขจากคนนอกได้

🔄

อัปเดตอัตโนมัติ

กด “ตรวจอัปเดต” ในโปรแกรม รับเวอร์ชันใหม่เอง

🌗

ปรับได้ตามใจ

ตั้งค่าความไวการอ่าน + ธีมสว่าง/มืด

คำถามที่พบบ่อย

ตรวจแล้วได้ 0 คะแนนทั้งใบ ทำไม?

มักเกิดจากเลือกฟอร์มผิด หรือภาพเอียง/กลับหัว — เลือกฟอร์มให้ตรงและสแกนใหม่ให้ตรง

โปรแกรมอ่านรอยฝนจางไม่เจอ

กด “⚙ ตั้งค่าความไว” แล้วลดค่าลง (จับรอยจางง่ายขึ้น) และแนะนำให้นักเรียนฝนเข้มขึ้นด้วยดินสอ 2B

ฝนสองช่องในข้อเดียวคิดคะแนนยังไง?

นับเป็น “ฝนซ้ำ” = 0 คะแนน และแสดงแยกให้เห็นชัด เช่นเดียวกับข้อที่ไม่ฝนเลย

ต้องต่อเน็ตไหม?

ตรวจข้อสอบใช้แบบออฟไลน์ได้ ต่อเน็ตเฉพาะตอนกด “ตรวจอัปเดต” เท่านั้น

ใช้กับ Mac / มือถือได้ไหม?

เวอร์ชันนี้สำหรับ Windows โดยดับเบิลคลิกไฟล์ .exe
